### Coding Conventions
- Use of ES6+ syntax and TypeScript features.
- Functional components and hooks for React.
- Utility-first styling with Tailwind CSS.

### Naming Patterns
- Component names use PascalCase.
- Hooks start with 'use' followed by camelCase.
- CSS classes follow BEM naming conventions when using @apply.

### Structure
- Co-located component files including styles and tests.
- Organized by feature/domain in both Next.js and Node.js.
